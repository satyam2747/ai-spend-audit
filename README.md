# 💸 SpendAudit: Is your startup burning money on AI?

Most startups overpay for AI tools without even realizing it. **SpendAudit** is a simple, no-nonsense tool that helps founders and teams find hidden savings in their AI stack in under 60 seconds.

Built for the **Credex** startup internship assignment.

## 🧐 Why SpendAudit?
You’re probably paying for ChatGPT Plus, Claude Pro, *and* a Cursor subscription. Do you really need all three? Probably not. 

SpendAudit looks at your tools, your team size, and your workflow to:
*   **Stop Over-Subscription:** Spot where you're paying twice for the same "intelligence."
*   **Optimize Plans:** Tell you if you're on a "Team" plan that's actually overkill for your size.
*   **Instant AI Summary:** Get a custom report generated by Claude (Anthropic) explaining exactly where to cut costs.

---

## 🚀 The Tech Behind the Savings
*   **Frontend:** A sleek, fast React app styled with Tailwind CSS (Modern Dark Mode).
*   **Backend:** A reliable Node.js/Express API.
*   **Database:** MongoDB Atlas to store and share audit results.
*   **The Brain:** Anthropic's Claude-3 Haiku for smart, personalized advice.
*   **Automation:** Nodemailer to send your audit report straight to your inbox.

## 🛠 Setting it up locally

### 1. Grab the code
```bash
git clone https://github.com/satyam2747/ai-spend-audit.git
cd ai-spend-audit
```

### 2. Get the engine running (Server)
```bash
cd server
npm install
# Create a .env file (use env_template.txt as a guide)
npm run dev
```

### 3. Launch the dashboard (Client)
```bash
cd ../client
npm install
npm run dev
```
