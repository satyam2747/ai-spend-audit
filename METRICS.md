# Metrics & KPIs

To measure the success of the SpendAudit tool as a lead-generation funnel, we track the following key performance indicators:

## 1. Funnel Metrics
- **Landing Page Visitors**: Total traffic to the frontend.
- **Audit Completion Rate**: Percentage of users who click "Start Audit" and successfully submit their tool stack. (Target: > 70%)
- **Lead Conversion Rate**: Percentage of users who provide their email address to view their full results. (Target: > 30%)

## 2. Value Metrics
- **Total Savings Identified**: Aggregate dollar amount of monthly/annual savings found by the platform across all users. 
- **Average Savings Per Startup**: The mean amount of money a startup can save (e.g., $150/month). Used for marketing copy.

## 3. Technical Metrics
- **AI Summary Generation Time**: Time taken for the Anthropic API to return the summary. Must be kept under 3 seconds to prevent user drop-off.
- **Error Rate**: Percentage of audits that fail due to API timeouts or server errors.

## 4. Growth Metrics
- **Viral Coefficient (K-factor)**: Number of new users generated by shareable result links (`/result/:id`).
